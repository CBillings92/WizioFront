Welcome to the Wizio API Guide!


What you'll need

With the Wizio API you'll be able to request and put Virtual Reality images right onto your own listing pages with ease. The first thing you will need is an API key/access token.

If you don't already have an API key/access token, please request more information on attaining one by clicking the button below and filling out the super brief form.




How utilizie the Wizio API

Currently the Wizio API utilizes an Iframe to play it's Virtual Reality player and the virtual reality images. The API works by placing a specific URL in the Iframe src attribute. This URL is dynamically generated by the Wizio API server upon a proper and validated request. The request must be a GET request to api.wizio.co.

Here is an example request - A full request will look like the following:

  • api.wizio.co/api/vr/listing/46a17cc1-3310-11e6-a0b2-0800274a1eca/175_amory_st_jamaica_plain_ma/2
  • api.wizio.co/api/vr/listing/{apikey}/{apartmentAddress}/{unitNumber}


The request can be broken up in the following way, in parameter order
  • /api/vr/listing/ --- this is a static part of the address. It will always be the same.
  • 6a17cc1-3310-11e6-a0b2-0800274a1eca --- This is an example API key that you will recieve from us.
  • 175_amory_st_jamaica_plain_ma--- This is the apartment address. The apartment address needs
    • Any spaces to be replaced with '_' (underscores)
    • A nearly complete address. The address does not need country, but it should have at least a street number, a street, a city/town and a state. A zip code is preferrable.



The response from the GET request will look like the following:
  • http://beta.wizio.co/listing/vr/6f6e7849-a6d5-46be-96a0-6aef59c5ce5d/08518c8d-327f-11e6-a0b2-0800274a1eca
This is the URL that must be placed into the src attribute of your Iframe. From here the Iframe will take over loading the rest.

Testing the API

Wizio has test data that you may use for testing to make sure the API is working. Below are the following addresses that test data exists for. To test, you'll want to make the request as noted above in the documentation with some of these hard coded addresses. The request from each of these responses will be a URL. You need to load this URL into the Iframe's src attribute on your page.

  • 101 Monmouth Street boston ma unit number 1
    • api.wizio.co/api/vr/listing/APIKEY/101_monmouth_st_boston_ma/1
