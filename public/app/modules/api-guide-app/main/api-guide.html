Wizio API Guide

Summray

The Wizio API allows you to display 360 content on your website dynamically.

Utilizing the Wizio API

The API utilizes iframes to display Wizio 360 tours. A request must be sent to Wizio's API with an API token, an address and a unit number to receive the dynamically generated link to the tour. Once the link to the tour is received it should be placed in an iframe src attribute.

Request

The request made to the Wizio API must be a GET request sent to https://api.wizio.co/tour/:apitoken/:address/:unitnumber

  • apitoken: Your API token provided by Wizio.
  • address: Address of the tour you're looking to pull.
  • unitnumber: Unit number of the tour you're looking to pull.

Response

The response from the Wizio API follows the following formats:


Success:

        {
            "contentURL": "https://www.wizio.co/tour/faebb92c-0502-4595-9b4a-12d0c93d05b3",
            "contentFound": true,
            "success": 1
        }
Failure:

        {
            "contentURL": "",
            "contentFound": false,
            "success": 0
        }

On a successful API request, the contentURL must be loaded into an iframe. From there the Wizio tour should load within the iframe.
